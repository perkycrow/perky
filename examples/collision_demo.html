Collision Demo

Demonstration of the 2D collision module integrated into the Perky framework

Controls:

  • WASD - Move the red player
  • Space - Jump
  • Click - Add a ball
  • R - Reset

This demo shows:

  • Automatic collisions between Canvas2D objects
  • Basic physics (gravity, bouncing, friction)
  • Static and dynamic objects
  • Visual debugging of collision shapes
