Tweakpane FPS Demo

Cette démo montre l'intégration de Tweakpane avec le framework Perky.

Le FPS graph utilise le plugin @tweakpane/plugin-essentials.

Contrôles Tweakpane :

  • FPS Graph : Monitoring en temps réel des performances
  • Animation Speed : Ajuster la vitesse d'animation
  • Particle Count : Nombre de particules à afficher
  • Debug Mode : Afficher/masquer les informations de debug
